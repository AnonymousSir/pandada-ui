# pandada-ui
仿照element-ui实现的UI库
## Project setup
```
npm i pandada-ui -s
```

## Usage
```
import Vue from 'vue'
import pandadaUI from 'pandada-ui'
Vue.use(pandadaUI)
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
